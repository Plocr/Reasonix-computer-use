# Excel/WPS 快捷键参考

仅在核心 Skill 的快捷键不足时读取。微软官方资料以 Excel Windows 桌面版为准；WPS 大多兼容，但执行后仍需检查 `changed`、选区或单元格值。

## 导航与选择

| 操作 | 快捷键 |
|---|---|
| 精确跳转 | 插件 `select_cell` / `select_range`，不要直接按 F5 |
| 移到当前数据区边缘 | `Ctrl+方向键` |
| 扩展到当前数据区边缘 | `Ctrl+Shift+方向键` |
| 工作表首格 | `Ctrl+Home` |
| 最后使用的单元格 | `Ctrl+End` |
| 扩展到首格 / 最后使用单元格 | `Ctrl+Shift+Home` / `Ctrl+Shift+End` |
| 整列 / 整行 | `Ctrl+Space` / `Shift+Space` |
| 切换工作表 | `Ctrl+PageUp` / `Ctrl+PageDown` |
| 在选区内前进 / 后退 | `Enter` / `Shift+Enter` |
| 在选区内向右 / 向左 | `Tab` / `Shift+Tab` |

## 编辑与填充

| 操作 | 快捷键 |
|---|---|
| 编辑活动单元格 | `F2` |
| 完成输入 | `Enter` |
| 取消输入 | `Esc` |
| 同值填满选区 | 输入后 `Ctrl+Enter` |
| 向下 / 向右填充 | `Ctrl+D` / `Ctrl+R` |
| 自动求和 | `Alt+=` |
| 快速填充 | `Ctrl+E`，WPS 支持情况需验证 |
| 插入单元格、行或列 | `Ctrl++`，属于结构修改，先确认 |
| 删除单元格、行或列 | `Ctrl+-`，必须确认 |
| 当前日期 / 时间 | `Ctrl+;` / `Ctrl+Shift+;` |

## 数据与公式

| 操作 | 快捷键 |
|---|---|
| 重新计算工作簿 | `F9` |
| 完整重新计算 | `Ctrl+Alt+F9` |
| 刷新外部数据 | `Alt+F5` |
| 刷新所有外部数据 | `Ctrl+Alt+F5` |
| 显示公式 | ``Ctrl+` `` |
